HTML Cheatsheet

A quick reference guide to the most common HTML elements.

Basics

Tag Name Description
<!DOCTYPE html> Document Type Declaration Should go at the top of any HTML file. Tells the browser that this document should be read as HTML. No closing tag.
<html> HTML A signal to the browser that the information within the tags is HTML code.
<head> Head Meta-information about this page.
<title> Title The title of the page (displayed on browser tab). Goes in head.
<body> Body Signals the body of the page (visible content).

Formatting

Tag Name Description
<h1-h6> Headings Designates heading text. H1-H6.
<p> Paragraph Designates paragraph text.
<br> Break Newline break (visual space).
<em> Emphasis - Italic Renders text italicized
<strong> Emphasis - Bold Renders text bold
<div> Division A generic stacked section designation in HTML (useful for CSS section selection).
<span> Span An inline division designation in HTML (useful for CSS section selection).

Lists

Tag Name Description
<ul> Unordered List Designates an unordered list.
<ol> Ordered List Designates an ordered list.
<li> List Item Designates an individual list item.

Other Elements

Tag Name Description
<a href="URL"> Link Anchor Turns wrapped text into a clickable link to the designated URL.
<img src="URL"> Image Inserts the designated image. Self-closing.
<video> Video Inserts the designated video. Requires a <source> tag
<embed src="URL"> Embedded Item Embeds the designated item. Self-closing.
<audio< Audio Inserts the designated audio file. Requires a <source>
<source src="URL"> Source File Used to designate the source media file, especially with <video> and <audio> tags.

Tables

Tag Name Description
<table> Table Designates a Table
<thead> Table Head Designates the top of the table (header row).
<th> Table Header Info Desginates content for a header row cell.
<tbody> Table Body Designates the body of the table.
<tr> Table Row Desginates a row of data in the table.
<td> Table Data Designates content for a table row cell.
<tfoot> Table Footer Designates the footer of the table. Often used to display column totals.

Useful Attributes

Attribute Name Description
alt="" Alternative Text Alternate descriptive text for an image or other element, useful to screenreaders or if an item fails to load.
src="URL" Source Link Designates which link an item should display. Often used with <img> and <source>
controls Controls Displays basic controls for videos and audio elements.
autoplay Autoplay Signals to start playing the element as soon as the page is loaded. Used with videos and audio files.
width Width Sets the width of an element. Often in pixels (px) and used with visual elements (images, videos, tables, etc)
height Height Sets the height of an element. Often in pixels (px) and used with visual elements (images, videos, tables, etc)
colspan=# Column Span Designates that the content should span multiple columns in a table. The number of columns is given by the # inserted after the =.
rowspan=# Row Span Designates that the content should span multiple rows in a table. The number of rows is given by the # inserted after the =.
